## üßê About <a name = "about"></a>

This project is a responsive HTML & CSS-based website designed for a fake animal shelter. Its goal is to provide a clean, user-friendly interface for visitors to browse adoptable pets, learn about upcoming events, and find information on donations.

The website includes multiple pages, including a home page, adoption page, pet gallery, events section, and donation information. It emphasizes accessibility, responsive design for different screen sizes, and visually appealing layouts to engage visitors and encourage animal adoption.

## üèÅ Getting Started <a name = "getting_started"></a>

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You only need a modern web browser to run the project locally:
Google Chrome, Mozilla Firefox, Safari, or Edge

Optionally, you can use a code editor like VS Code for development:
https://code.visualstudio.com/

### Installing

1. Clone the repository to your local machine:

```
git clone https://github.com/ElinaTsfr/animal-shelter.git
```

2. Open the project folder in your code editor or file explorer:

```
cd animal-shelter
```

3. Open `index.html` in your web browser to view the homepage.
   Double-click index.html or use the browser's "Open File" option

## üé® Design Choices <a name="design_choices"></a>

The design of this animal shelter website was carefully chosen to enhance usability, accessibility, and engagement:

- **Color Palette:** Soft, neutral, and warm tones were used to create a welcoming and friendly atmosphere for visitors.
- **Typography:** Clear, readable fonts with consistent sizing were chosen to improve legibility across devices.
- **Layouts:** Flexbox and CSS Grid were used extensively to build responsive layouts for the homepage, gallery, and event sections.
- **Card-Based Design:** Adoptable pets and events are displayed in card-style layouts for easy scanning and visual appeal.
- **Interactive Elements:** Subtle hover effects and transitions provide interactivity without JavaScript.
- **Responsive Design:** Media queries ensure the website adapts seamlessly to desktop, tablet, and mobile screens.
- **Accessibility:** High contrast text, descriptive alt attributes for images, and clear navigation were implemented for an inclusive user experience.
